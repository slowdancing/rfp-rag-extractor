## ✅ 검색 품질 — 해결

**핵심 발견 (실험으로 검증)**: 검색 실패의 원인은 **청크 크기가 아니라 질의 표현**.
- 청크 256~1024 비교 → 모두 Hit@1 0.89~0.93, **1024 최적** (가설 반증)
- 자연어 질문은 흔한 일반어(사업·예산·기간)가 핵심어를 희석

**조치 (완료)**:
- [x] 질의 재작성(Query Rewriting) → 실패 질의 순위 20위 밖 → 12위
- [x] **하이브리드 검색(BM25 + dense, RRF)** → 실패 질의 **14위 밖 → 1위** ★
- [ ] 사업명 가중 인덱싱 / 재랭킹 (보조, 선택)

**검증**: end-to-end 재테스트에서 어제 실패 질문("~사업규모와 과업기간은?")이
이제 정확히 답변(87,000,000원 / 120일 + 출처). 상세: `results/retrieval_experiment.md`
