## 🟡 검색 품질 — 진행 상황

**핵심 발견 (실험으로 검증)**: 검색 실패의 원인은 **청크 크기가 아니라 질의 표현**.
- 청크 256~1024 비교 → 모두 Hit@1 0.89~0.93, **1024 최적** (가설 반증)
- 같은 문서도 질의에 따라 순위 급변: 사업요약 1위 / 사업명 2위 / 자연어 질문 20위 밖
- 자연어 질문은 흔한 일반어(사업·예산·기간)가 핵심어를 희석

**조치**:
- [x] 질의 재작성(Query Rewriting) 구현 → 실패 질의 순위 **20위 밖 → 12위** (부분 개선)
- [ ] 하이브리드 검색(BM25 + dense) — 고유명사 정확 매칭 (다음 우선순위)
- [ ] 사업명 가중 인덱싱 / 재랭킹 (보조)
